Build Your Own Plugin

Oxidian plugins are JavaScript modules with a simple manifest. If you've written an Obsidian plugin, you already know how.

Plugin Manifest ‚Äî manifest.json

Every plugin needs a manifest.json in its root directory:

{ "id": "my-plugin", "name": "My Plugin", "version": "1.0.0", "author": "YourName", "description": "What it does", "main": "main.js", "minOxidianVersion": "1.0.0" }

Basic Plugin Structure ‚Äî main.js

const { Plugin } = require('oxidian'); class MyPlugin extends Plugin { onload() { // Register a command this.addCommand({ id: 'say-hello', name: 'Say Hello', callback: () => { new Notice('Hello from my plugin!'); } }); // Add a settings tab this.addSettingTab(new MySettingTab(this.app, this)); // Register a custom view this.registerView('my-view', (leaf) => new MyView(leaf)); console.log('MyPlugin loaded!'); } onunload() { console.log('MyPlugin unloaded.'); } } module.exports = MyPlugin;

API Reference

MethodDescription
addCommand(cmd)Register a command in the palette. cmd has id, name, callback, optional hotkeys.
addSettingTab(tab)Add a settings tab. Extend PluginSettingTab and implement display().
registerView(type, factory)Register a custom view type. Factory receives a WorkspaceLeaf.
addRibbonIcon(icon, title, cb)Add an icon to the left ribbon sidebar.
addStatusBarItem()Returns an HTMLElement appended to the status bar.
registerEvent(ref)Auto-unregister an event listener on plugin unload.
registerInterval(id)Auto-clear a setInterval on unload.
loadData()Load persisted plugin data (returns Promise<any>).
saveData(data)Persist plugin data as JSON.
app.vault.read(file)Read a file's contents as string.
app.vault.modify(file, data)Write new contents to a file.
app.vault.create(path, data)Create a new file in the vault.

1

Fork

Fork the oxidian/plugin-registry repo on GitHub.

2

Add Manifest

Add your manifest.json and repo URL to the registry.

3

Submit PR

Open a Pull Request. We'll review within 48 hours.

4

Published!

Once merged, your plugin appears in the Plugin Explorer.
