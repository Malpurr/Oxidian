API Reference

Complete reference of Oxidian's Tauri IPC commands ‚Äî the bridge between frontend and Rust backend.

File Operations #

read_file

Read the contents of a file from the vault.

typescript
invoke('read_file', { path: string }): Promise<string>
ParameterTypeDescription
pathstringRelative path within the vault

save_file

Write content to a file. Creates the file if it doesn't exist.

typescript
invoke('save_file', { path: string, content: string }): Promise<void>

delete_file

Delete a file from the vault.

typescript
invoke('delete_file', { path: string }): Promise<void>

rename_file

Rename or move a file within the vault. Automatically updates wiki-links.

typescript
invoke('rename_file', { oldPath: string, newPath: string }): Promise<void>

list_files

List all files in the vault or a subdirectory.

typescript
invoke('list_files', { path?: string }): Promise<FileEntry[]>

create_folder

Create a new folder in the vault.

typescript
invoke('create_folder', { path: string }): Promise<void>

Search #

search

Full-text search across the vault using Tantivy.

typescript
invoke('search', { query: string, limit?: number }): Promise<SearchResult[]>

interface SearchResult {
  path: string;
  title: string;
  snippet: string;
  score: number;
}

rebuild_search_index

Force a full re-index of the vault.

typescript
invoke('rebuild_search_index'): Promise<void>

decrypt_vault

typescript
invoke('decrypt_vault', { password: string }): Promise<void>

Events #

Listen for backend events using Tauri's event system:

javascript
import { listen } from '@tauri-apps/api/event';

// File changes
await listen('file-changed', (event) => {
  console.log('File changed:', event.payload.path);
});

// Search index updated
await listen('index-updated', (event) => {
  console.log('Index updated, files:', event.payload.count);
});
EventPayloadDescription
file-changed{ path, kind }File created, modified, or deleted
index-updated{ count }Search index rebuilt
vault-locked{}Vault was locked
vault-unlocked{}Vault was unlocked
plugin-loaded{ id, name }Plugin finished loading
